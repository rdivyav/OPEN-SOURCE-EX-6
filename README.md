# OPEN-SOURCE-EX-6
# Aim:
To create a new user account named alies with a specific user ID (UID 1326) on a Linux (RHEL 9) system.
# Procedure:
1. Log in as the root user
Administrative privileges are required to create new users.
2. Create the user account alies with UID 1326
Use the user creation command with the UID option.
3. Verify that the user account was created successfully
Check the /etc/passwd file or use user information commands.
